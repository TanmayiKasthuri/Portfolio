Some Things I’ve Built


EverydayCook - The Recipe Finder

React, Django, Sqlite, AWS S3, AWS EC2

EverydayCook an innovative recipe finder application developed using Django, React and AWS. This application enhances the cooking experience by allowing users to input available ingredients and receive dish recommendations, acting as an intelligent kitchen assistant. Additionally, EverydayCook serves as an electronic cookbook, featuring a curated collection of popular recipes complete with detailed instructions and precise cooking times to ensure perfect results. The backend REST APIs were built with Django, while the frontend was crafted using React. The application was successfully deployed leveraging AWS services, including EC2 and S3, to ensure scalability and reliability.

FitConnect : Your Social Fitness Hub

ASP .NET Core MVC, Sql Server, Bootstrap

Designed a web-based social media application for fitness enthusiasts, enhancing user experience by efficiently tracking locations and delivering personalized recommendations. The app provides tailored suggestions for races and clubs based on user preferences and activity levels. Integrated advanced geolocation features to track user movements in real-time and offer relevant fitness activities nearby. This project aimed to foster a community of fitness enthusiasts by enabling users to discover and participate in local fitness events and clubs, ultimately promoting a healthier and more active lifestyle.

EasyConsult - Doctor Patient Consultation Redefined

React, Node JS, Mongo DB, Express JS, MERN Stack

This is a cutting-edge healthcare communication solution named Easyconsult. Leveraging MongoDB Atlas for a robust database, and building RESTful API endpoints with Express and Node.js, I created a seamless real-time interaction platform for doctors and patients using ReactJS on the frontend. The application employs WebSocket communication for live updates during consultations. Easyconsult features dynamic QR code generation for consultations, transparent disease/condition selection with medication details, secure user authentication, and consultation review capabilities. This innovative two-pronged app fosters enhanced doctor-patient communication, improving adherence to medical instructions and patient empowerment.

ShopIt

Django framework, Python, Bootstrap, HTML, CSS

With a commitment to promoting eco-conscious buying and selling, ShopIt provides a seamless online marketplace where users can effortlessly browse and purchase environmentally-friendly products. Harnessing the Django ORM (Object-Relational Mapping) capabilities, ShopIt ensures efficient data management and seamless interactions between the backend and frontend. Under the hood, PostgreSQL serves as the backbone of our database, ensuring reliability, scalability, and performance. With PGAdmin as our user-friendly interface, database management becomes a breeze and optimizes data operations

Gator Taxi

Advanced Data Structures

"Gator Taxi" is a widely used ride-sharing service that addresses the challenge of managing a large volume of ride requests. It represents an innovative software solution capable of efficiently managing and tracking ride requests. To meet this demand, a new software program was developed using Python. The program utilizes advanced data structures to efficiently handle the influx of ride requests, ensuring seamless operation and optimal user experience. Through its intuitive design and robust functionality, "Gator Taxi" streamlines the process of requesting and managing rides, making it a preferred choice for both riders and drivers in the transportation industry.

OpenWeather API

JavaScript, HTML, CSS

Implemented a weather application leveraging the OpenWeather API to provide real-time weather information for any specified city. Built with JavaScript, HTML, and CSS, this app offers a seamless user experience, allowing users to easily search for and view current weather conditions, including temperature, humidity, and weather descriptions. By fetching data directly from the API, the app ensures accuracy and up-to-date information. Despite using simple and foundational web technologies, this application exemplifies the powerful role APIs play in aggregating and disseminating global weather data, transforming basic code into a tool that connects users with the world's climate information at their fingertips.
